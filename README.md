# Social Engine (Demo)

## Overview   

- Read-only interactive demo of Social_Engine 
- A personal CRM built for relationship tracking. 
- Showcases the UI/UX including the registry, heat-decay system, and calendar. Full version connects to Supabase with auth, real-time data, and persistent storage.
---
### Here's what's included:
1. Home Dashboard - 7 mock active units with heat scores, category chips, search/filter, sort modes (Recent/Decay), and a quick-add slot. The Schedule button shows an upcoming event badge.
2. Active Registry - full list view with heat bars, last interaction dates, archive button per unit, and "Register New Unit" to add people on the fly.
3. Archive Vault - 1 archived unit pre-loaded (Haris Malik). Restore or permanently purge.
4. Calendar - fully interactive 3-column layout: grid navigator, day detail panel with add/delete events, and an upcoming sidebar. Events persist across the demo session.
5. Unit Modal - clicking any person opens their full profile with interaction logging (Texted/Called/Meet Up chips), follow-up scheduler, and editable profile fields via the Edit button.
6. All data changes (adding people, logging interactions, archiving, adding calendar events) persist within the session but reset on page reload - exactly like a demo should behave.
